📘 Adaptive Learning System

A smart full-stack web application that helps students prioritize their study topics based on difficulty and understanding level.

---

🚀 Features

- Add, delete, and manage study topics
- Generate priority-based study plans
- Real-time updates using JavaScript
- Backend APIs using Node.js & Express
- MongoDB integration for persistent storage

---

🛠 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB

---

⚙️ How It Works

Each topic is assigned a priority score based on:

- Difficulty (Easy / Medium / Hard)
- Understanding (Good / Average / Bad)

Higher score ⇒ Higher priority in study plan

---

▶️ Run Locally

cd backend

npm install

node server.js

Open "frontend/index.html" in browser
